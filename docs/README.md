#### 1. 我們收集的信息

在您使用我们游戏服务的过程中，我们会按照如下方式收集您在使用服务时主动提供的或因为使用服务而产生的信息，用以向您提供、优化我们的服务以及保障您的账户安全：

**1.1 账号注册与登录**
当您注册或使用游戏服务时，我们使用 **TapTap 登录服务**。我们会收集您的 **TapTap OpenID、昵称、头像**。

*   **收集目的**：用于标记您为我们的用户、创建游戏内部账号以及保存您的游戏进度（云存档）。

**1.2 实名认证（防沉迷）**
为满足中国大陆《关于进一步严格管理 切实防止未成年人沉迷网络游戏的通知》等相关法律法规政策及主管部门的要求，游戏用户需进行实名认证。我们通过 **TapTap 提供的实名认证系统**校验您的身份。
*   **特别说明**：实名身份信息（包括真实姓名、身份证号）属于**个人敏感信息**。该信息将由 TapTap 直接收集并校验，**我们不会将您的真实姓名和身份证号存储在我们的服务器上**。拒绝提供实名身份信息可能会导致您无法登录游戏或受到防沉迷系统的限制。

**1.3 设备与网络信息（用于游戏适配与安全）**
本游戏基于 **团结引擎** 开发。为了确保游戏能在您的设备上正常运行（适配屏幕分辨率、处理器性能等）以及保障账号安全，我们会收集您的以下信息：

*   **设备信息**：设备型号、操作系统版本、系统语言、GPU 信息、内存容量、屏幕分辨率、设备唯一标识符（如 Android ID / OAID）。
*   **网络信息**：网络类型（WiFi/4G/5G）、网络连接状态。
*   **收集目的**：上述信息用于游戏画面的渲染适配、Crash（崩溃）问题的排查以及识别异常设备以打击作弊行为。

**1.4 游戏日志与服务优化（自有数据分析）**
为了提升游戏体验并修复可能的故障，我们开发了内部的数据埋点系统。当您游玩时，我们会收集您的**游戏日志信息**：
*   **内容包括**：登录时间、关卡进度、物品获取与消耗记录、操作路径、游戏崩溃堆栈信息。
*   **使用方式**：上述信息将加密上传至我们控制的服务器，**仅用于游戏运营统计分析（如关卡难度平衡）、优化游戏数值体验及修复程序 Bug**，不会用于广告画像，也不会出售给任何无关第三方。

**1.5 权限调用说明**
我们在提供服务过程中，可能会调用您设备的以下权限。请您知悉，我们不会默认开启这些权限，仅在您同意的情况下才会调用：
*   **存储权限（读写外部存储）**：用于将游戏配置信息保存在您的设备本地，以确保断网情况下也能正常读取存档。
*   **网络权限**：用于上传上述提到的游戏优化数据、验证登录状态以及同步云端存档。
*   **特别声明**：本游戏**不包含**相机、麦克风、电话拨打、短信发送及地理位置定位功能，我们**不会**申请也不会调用上述敏感权限。

**1.6 征得授权同意的例外**
根据相关法律法规及国家标准，以下情形中，我们收集、使用您的相关个人信息**无需**征求您的授权同意：
1.  与国家安全、国防安全直接相关的；
2.  与公共安全、公共卫生、重大公共利益直接相关的；
3.  与刑事侦查、起诉、审判和判决执行等直接相关的；
4.  出于维护您或其他个人的生命、财产等重大合法权益但又很难得到本人同意的；
5.  所收集的个人信息是您自行向社会公众公开的；
6.  从合法公开披露的信息中收集个人信息的，如合法的新闻报道、政府信息公开等渠道；
7.  根据您要求签订和履行合同所必需的；
8.  用于维护所提供的产品或服务的安全稳定运行所必需的，例如发现、处置产品或服务的故障。

#### 2. 信息的存储

**2.1 信息存储的地点**
我们是一家位于**中国香港**的游戏工作室。

*   **本地存储**：您的设置偏好等数据将直接存储在您的**移动设备本地**，这部分数据由您自行管理。
*   **服务器存储**：为了优化游戏体验，我们收集的非敏感游戏日志（如游戏崩溃信息、关卡通关数据）将上传至我们控制的服务器进行处理。
*   **跨境传输说明**：由于我们的运营主体位于中国香港，您知悉并同意，在符合中国大陆相关法律法规（如《个人信息保护法》）的前提下，上述非敏感的游戏运营数据可能会被传输至中国香港进行处理。我们将采取严格的安全措施，确保您的个人信息得到与中国大陆法律法规同等水平的保护。

**2.2 信息存储的期限**
我们只会在为实现本政策所述目的所必需的时间内，或在法律法规规定的期限内保留您的个人信息：
*   **游戏日志数据**：我们将保留您的游戏操作日志及统计数据，直至您注销账号或我们停止游戏运营。
*   **账号信息**：如果您申请注销账号，我们将会在完成注销流程后，对您的账号信息及相关游戏数据进行**删除**或**匿名化处理**。
*   **例外情况**：在法律法规要求（如《电子商务法》要求交易信息保存三年）或配合公安机关调查等特定情况下，我们可能会在法律规定的范围内延长相关信息的存储期限。

**2.3 产品或服务停止运营时的通知**
如果未来出现我们不得不停止运营本游戏的情况，我们将严格遵守法律法规的要求：

1.  我们将至少提前 **60 天** 通过游戏内公告、官方社区公告或推送通知等形式通知您；
2.  停止运营后，我们将**停止收集**您的个人信息；
3.  对于已收集的个人信息，我们将在法律法规规定的期限内进行**删除**或**匿名化处理**，保障您的信息安全。

#### 3. 信息安全与风险提示

**3.1 我们采取的安全保护措施**
我们非常重视您的个人信息安全，并采取一切合理可行的技术和管理措施来保护您的个人信息：

- **技术保护手段**：我们使用业界通用的安全技术来防止信息遭到未经授权的访问、披露、使用或修改。例如，我们将使用加密技术（如 HTTPS/SSL 协议）确保您游戏日志数据在传输过程中的安全性；对于服务器端存储的数据，我们会采用权限控制机制。
- **访问控制**：我们严格限制访问信息的人员范围。仅有为实现游戏优化、运营分析目的而有必要知晓这些信息的开发人员才能接触到相关数据，并要求其遵守保密义务。
- **数据安全能力**：我们会定期检查系统的安全策略，确保存储和处理数据的系统始终处于受保护的状态。

**3.2 安全事件处置**

- 尽管我们采取了上述安全措施，但请您理解，在互联网行业，由于技术的限制和飞速发展，以及可能存在的各种恶意攻击手段，即便竭尽所能加强安全措施，也不可能始终保证信息百分之百的安全。
- 在不幸发生个人信息安全事件（如数据泄露）后，我们将严格按照法律法规的要求，及时向您告知：安全事件的基本情况和可能的影响、我们已采取或将要采取的处置措施、您可自主防范和降低风险的建议、对您的补救措施等。
- 我们将通过游戏内公告、推送通知或官方社区公告等方式告知您事件相关情况。难以逐一告知个人信息主体时，我们会采取合理、有效的方式发布公告。

**3.3 风险提示与免责条款**

- **不可抗力**：如果我们的物理、技术或管理防护设施遭到破坏（如遭遇黑客恶意攻击、病毒爆发、电力故障等不可抗力因素），导致信息被非授权访问、公开披露、篡改或毁坏，在我们将风险降到最低并已履行法律规定的告知义务的前提下，我们将在法律允许的范围内免除相应的责任。
- **第三方风险**：请您知悉，如果您通过本游戏中的链接跳转到第三方提供的服务（如 TapTap 社区页面），该第三方服务由其独立运营，不受本政策约束。请您在向第三方提供个人信息前，仔细阅读其隐私政策。

**3.4 您的自我保护责任**

- 请您妥善保管您的游戏账号及 TapTap 登录信息。**请勿将您的账号转让、租借或分享给他人使用**。
- 若您发现您的账号被他人非法使用或存在安全漏洞，请立即通过本政策“联系我们”章节中提供的联系方式通知我们，以便我们协助您采取暂停账号登录等措施。

我们会严格遵守法律法规及本政策的约定，将收集的信息用于以下目的。若我们将您的信息用于本政策未载明的其他用途，或者将基于特定目的收集而来的信息用于其他目的，我们将事先征求您的同意。

1. **提供基础游戏服务**：利用您的设备权限和本地存储数据，实现游戏的启动、画面渲染、存档保存与读取等核心功能。
2. **产品开发与服务优化**：利用收集的游戏日志（如关卡通过率、死亡地点统计），分析游戏数值设计的合理性，优化游戏难度曲线，并修复程序中的崩溃与漏洞。
3. **身份认证与安全保障**：利用 TapTap SDK 提供的实名认证服务，履行防沉迷法定义务；利用设备标识符识别异常的高频请求或作弊行为，维护公平的游戏环境。

**（1）TapTap 登录与防沉迷 SDK**

- **第三方公司名称**：易玩（上海）网络科技有限公司
- **使用目的**：提供 TapTap 账号一键登录功能、验证用户身份、执行未成年人防沉迷限制。
- **涉及个人信息**：设备信息（Android ID、OAID、设备型号、操作系统版本）、TapTap 账号信息（OpenID、昵称、头像）、应用列表（仅用于判断是否安装了 TapTap 客户端以唤起登录）。

**4.3 特别声明**

#### 5. 对外提供

**5.1 共享**
我们严格限制个人信息的共享。除以下情形外，我们不会向任何公司、组织或个人共享您的个人信息：

1. **在获取明确同意的情况下共享**：获得您的明确同意后，我们会与其他方共享您的个人信息。
2. **在法定情形下的共享**：我们可能会根据法律法规规定、诉讼争议解决需要，或按行政、司法机关依法提出的要求，对外共享您的个人信息。
3. **与授权合作伙伴共享（SDK）**：仅为实现本政策中声明的目的，我们的某些服务将由授权合作伙伴提供（即上文提到的 **TapTap**）。我们仅会出于合法、正当、必要、特定、明确的目的共享您的信息，并且只会共享提供服务所必要的信息。我们的合作伙伴无权将共享的个人信息用于任何其他用途。

**5.2 转让**
我们不会将您的个人信息转让给任何公司、组织和个人，但以下情况除外：

1. **获得您的明确同意**：在获得您的明确同意后，我们会向其他方转让您的个人信息。
2. **涉及合并、收购或破产清算**：随着我们业务的发展，如果我们（霁曜游戏工作室）涉及合并、收购、资产转让或类似的交易，而您的信息有可能作为此类交易的一部分而被转移。我们将要求新的持有您个人信息的公司、组织继续受本政策的约束，否则我们将要求该公司、组织重新向您征求授权同意。

**5.3 公开披露**
我们仅会在以下情况下，且采取符合业界标准的安全防护措施的前提下，才会公开披露您的个人信息：

1. **获得您的明确同意**；
2. **基于法律的披露**：在法律、法律程序、诉讼或政府主管部门强制性要求的情况下，我们可能会公开披露您的个人信息。
3. **违规处罚公告**：为维护游戏及其用户的合法权益，若您违反游戏规则（如使用外挂、作弊），我们可能会在对您的账号ID、昵称进行**去标识化/掩码处理**（例如：玩家 张*三，ID：12***56）后，在公告中展示相关违规信息。

**5.4 共享、转让、公开披露个人信息时事先征得授权同意的例外**
根据相关法律法规，以下情形中，我们共享、转让、公开披露您的个人信息**无需**事先征得您的授权同意：

1. 与国家安全、国防安全直接相关的；
2. 与公共安全、公共卫生、重大公共利益直接相关的；
3. 与刑事侦起诉、审判和判决执行等直接相关的；
4. 出于维护您或其他个人的生命、财产等重大合法权益但又很难得到本人同意的；
5. 您自行向社会公众公开的个人信息；
6. 从合法公开披露的信息中收集个人信息的，如合法的新闻报道、政府信息公开等渠道。

### 6. 您的权利

在您使用本游戏期间，我们严格遵守法律法规的规定，保障您对自己的个人信息行使以下权利：

**6.1 查阅、复制与更正您的信息**

- **账号注册信息**：由于本游戏使用 **TapTap 账号**登录，您的头像、昵称、密码等基础注册信息由 TapTap 平台管理。如需更正或修改这些信息，请您登录 TapTap 客户端或网页版进行操作，数据同步可能需要重新登录游戏才能生效。
- **获取信息副本**：如您需要获取我们收集的关于您的个人信息副本（如游戏日志数据），您可以用本政策第 8 条提供的联系方式与我们联系。

**6.2 删除个人信息与注销账号**
我们提供账号注销和个人信息删除的渠道。由于本游戏未设置独立的账号安全中心，您可以按照以下方式申请注销：

1. **申请方式**：请使用您注册/登录游戏时的关联邮箱或联系方式，发送电子邮件至本政策列明的联系邮箱。
2. **邮件内容**：邮件标题请注明【申请注销游戏账号】，并在正文中提供您的游戏 UID（可在游戏设置界面查看）或 TapTap OpenID 截图，以便我们验证您的身份。
3. **处理流程**：我们在收到您的申请并验证身份无误后，将在 **15 个工作日**内完成注销处理。
4. **注销后果**：注销完成后，我们将停止为您提供产品服务，并依据您的要求，**删除**您在服务器上的所有游戏数据（包括云存档、游戏日志）或对其进行**匿名化处理**。 **请您知悉，注销是不可逆的操作，数据一旦删除将无法恢复。 **

**6.3 改变或撤回授权范围**

- **系统权限**：您可以通过移动设备的「设置」功能（如：设置-应用管理-权限管理），关闭您此前授权的「存储」或「网络」权限。请注意，关闭某些必要权限可能导致游戏无法正常运行（例如无法保存游戏进度）。
- **撤回隐私授权**：如果您不同意本隐私政策的内容，您可以选择停止使用本游戏。如果您希望撤回对本隐私政策的整体授权，您可以通过本政策注明的联系方式联系我们注销账号，届时我们将停止处理您的个人信息。

#### 7. 未成年人保护
